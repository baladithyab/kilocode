# Evolution Layer: 5-Minute Quick Start

Get started with the Evolution Layer in minutes using the Settings UI.

ðŸ’¡ Tip: Start with **Level 0** on your first day. You'll learn the proposal workflow without any auto-application.

## Step 1: Bootstrap (30 seconds)

1. Open VS Code Settings
2. Navigate to "Evolution" tab
3. Click "Bootstrap Evolution Layer"

## Step 2: Configure Council (1 minute)

1. In the Council Configuration section
2. Select 3-5 profiles from the dropdown
3. Click "Configure Council"

## Step 3: Set Automation Level (30 seconds)

1. In the Automation Level section
2. Choose your level:
    - **Level 0**: Manual review (recommended for first-time users)
    - **Level 1**: Auto-triggered reviews, manual approval
    - **Level 2**: Auto-apply safe changes
    - **Level 3**: Full automation
3. Click "Save Configuration"

## Step 4: Complete a Task (2 minutes)

1. Complete any coding task with Kilo Code
2. In Evolution Settings, click "Export Trace"
3. Click "Run Council Review"
4. Review the generated proposal
